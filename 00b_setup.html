PROLOGUE · WHY BOTHER?

Why prepare an environment first?

Writing Python needs two things: an interpreter that executes your code, and a place to write it and see the results. All materials in this course are .ipynb Jupyter notebooks — documents that mix explanations, runnable code, and outputs, perfect for studying alongside these companion pages. Since you study at home, pick whichever of the three options below fits you — one is enough:

OPTION 1 · ZERO INSTALL · RECOMMENDED

Google Colab — Python in your browser

Google Colab is a free online Jupyter Notebook service: open the browser, sign in with a Gmail account, and write Python — nothing to install, and your work follows you to any computer.

Steps

  1. Get a Gmail account, then open Google Colab and sign in.
  2. Download the course notebooks (.ipynb files) from 網路大學, then upload them to your Google Drive.
  3. In Google Drive, right-click a notebook → Open with → Google Colaboratory — start reading and running cells.
✔ EVERY COURSE LIBRARY IS PRE-INSTALLED

Colab already ships with all five libraries this course uses — NumPy, Matplotlib, SymPy, SciPy, and Pandas — so every notebook from Module 1 to Module 5 runs out of the box. (Each course notebook also starts with a package-check cell that installs anything missing.)

⚠ GOOD TO KNOW

Notebooks opened from Drive are auto-saved back to Drive, like Google Docs. The free tier is plenty for this course — no GPU needed. If a session sits idle it disconnects; just reconnect and re-run the cells (Runtime → Run all).

OPTION 2 · LOCAL INSTALL

Anaconda + Jupyter — your own machine, works offline

Anaconda is a one-shot installer that sets up the Python interpreter, Jupyter Notebook, and the entire scientific stack of this course (NumPy, Matplotlib, SymPy, SciPy, Pandas — all included). Install once and you can study offline, forever.

Steps

  1. Download Anaconda for your OS (Windows / macOS) and install it.
  2. Download the course notebooks from 網路大學 and unzip them into a folder you can find again.
  3. Launch Jupyter Notebook (from Anaconda Navigator or by running jupyter notebook in a terminal), browse to the course folder, and open a notebook such as 01_Basic_NumPy.ipynb.
⚠ COMMON HICCUPS

The installer is large (500 MB+): allow 10–20 minutes and ~5 GB of disk. If the download stalls, antivirus software or a restrictive network is the usual culprit — try another network. On Windows, prefer "Just Me" installation if you hit permission errors.

🎞 OPTIONAL · SLIDE MODE

The course notebooks are also designed for RISE (a Jupyter extension that turns notebooks into presentation slides). Installing RISE is optional — the static slide HTML files on 網路大學 show the same content without any setup.

OPTION 3 · LIGHTWEIGHT

VS Code or pip — for those who already have Python

Already comfortable with a terminal and an editor? Two lighter paths:

A. VS Code + Jupyter extension

  1. Install VS Code, then add the Python and Jupyter extensions.
  2. Open any course .ipynb file directly — VS Code runs it cell by cell just like Jupyter.

B. Plain pip in a virtual environment

python -m venv scicomp # create an isolated environment source scicomp/bin/activate # Windows: scicomp\Scripts\activate pip install numpy matplotlib sympy scipy pandas jupyter jupyter notebook # then open the course notebooks

Whichever option you chose, verify it with this two-liner in a notebook cell — every version should print without an error:

import numpy, matplotlib, sympy, scipy, pandas print(numpy.__version__, matplotlib.__version__, sympy.__version__, scipy.__version__, pandas.__version__)
PART 04 · YOUR AI TUTOR

Colab's Gemini "Learn Mode" — an AI that teaches instead of ghostwriting

If you picked Google Colab, one feature deserves special attention. Every Colab notebook has a built-in Gemini chat, and Google added two companion features aimed squarely at self-learners: Custom Instructions and Learn Mode.

FeatureWhat it does
Custom InstructionsStore your preferences — your style rules, which module you're on, "answer using only methods we've covered" — at the notebook level. The instructions travel with the notebook, so a study group can share one pre-configured assistant.
Learn ModePowered by Custom Instructions. When on, Gemini stops handing you finished code to copy-paste. Instead it answers with step-by-step guidance that breaks down the concept — like a tutor beside you, not a ghostwriter. Toggle it in the Gemini chat window.
✦ WHY THIS COURSE RECOMMENDS IT

Learn Mode is built for exactly the situation you are in — self-studying without a teacher in the room. When you hit code you don't understand, asking "why is it written this way?" in Learn Mode feels like being guided step by step — a perfect match for this course's predict → run → verify habit, and for everything Chapter 00A said about protecting your learning loop.

Google provides official example notebooks with Learn Mode pre-configured — open one and feel how it guides rather than answers:

🤖 PROMPTS THAT WORK WELL IN LEARN MODE
Why does this cell output this result? Walk me through it — don't give me the final answer yet. How does this relate to the views-vs-copies idea from Module 1.3? (give the AI your course context) Don't write the code for me — guide me step by step and let me type each part myself.

If your Colab UI looks different, Google may still be rolling the feature out to your account — study normally and check again later. Source: Google's official blog: Introducing Learn Mode in Google Colab.

FAQ

⚠ Q & A

Q: Can I use both Colab and a local install?
Yes — they don't interfere. Many students use Anaconda at home and Colab everywhere else.

Q: Will my edits in Colab be lost?
No. Notebooks opened from Google Drive are saved back to Drive automatically, like Google Docs.

Q: I can't find the Gemini chat window.
It normally sits at the top-right or in the side panel of a Colab notebook. If it's absent, the feature may not have reached your account yet — everything else in the course works without it.

Q: Do I need a GPU or Colab Pro?
No. Every notebook in Modules 1–5 runs comfortably on the free CPU tier.
